Preservar a liberdade de instalar o que você quiser

Você é o único a decidir o que é ou não está instalado no seu telefone, laptop ou outro dispositivo. Se você comprar um dispositivo eletrônico de uma empresa, eles não devem forçar você a ter aplicativos ou softwares específicos instalados nele.

Como funciona

Por que Liberdade de instalação ?

Para muitos produtos como um laptop ou um smartphone Android, faz sentido que você decida o que é executado no seu computador. No entanto, alguns smartphones contêm aplicativos que você não pode desinstalar - mas eles ainda ocupam memória, espaço de armazenamento e às vezes até mesmo diminuem o dispositivo que você comprou.

Durante a pandemia, muitas universidades começaram a exigir que os alunos instalem o software procritor em seus dispositivos para monitorar seus laptops durante os exames. Da mesma forma, plataformas como discórdia e whatsapp banir usuários que escrevem "auto-bots" porque não usam o aplicativo padrão.

Como podemos consertar isso?

Nossa mentalidade precisa mudar de que você deve sempre ser capaz de executar o código que você confia. Se você não quiser executar qualquer software apresentado, você pode usá-lo ou encontrar (ou criar) uma alternativa a si mesmo.

Organizações e empresas podem fornecer software cliente que seus usuários podem ser executados, mas usar esse software nunca deve ser obrigado ou necessário. Deve ser sempre possível que os usuários escrevam um cliente alternativo que interage com um servidor da mesma maneira.

Direitos semelhantes

Autonomia de software.

Jardins murados limitam a liberdade. Qualquer um deve ter o direito de reprogramar seu dispositivo.

Continuidade do abandono

Ao contrário do Mickey Mouse, não poder usar software de uma pessoa morta pode prejudicar os humanos. Portanto, o software deve se tornar o domínio público no final do seu ciclo de vida.

Garantia de software

O software não corra de causas naturais, merece uma expectativa de vida mais longa.
