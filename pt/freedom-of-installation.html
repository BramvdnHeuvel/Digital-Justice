Preserve a liberdade de instalação o que quiser

É o único a decidir o que está ou não instalado no seu telefone, portátil ou outro dispositivo. Se comprar um dispositivo electrónico a uma empresa, eles não devem poder forçá-lo a ter aplicações ou software específicos instalados no mesmo.

Como funciona

Porque liberdade de instalação?

Para muitos produtos como um portátil ou um smartphone Android, faz sentido que seja o utilizador a decidir o que corre no seu computador. No entanto, alguns smartphones contêm aplicações que não pode desinstalar - mas ainda assim ocupam memória, espaço de armazenamento e por vezes até tornam mais lento o dispositivo que comprou.

Durante a pandemia, muitas universidades começaram a exigir aos estudantes a instalação de software de proctoring nos seus dispositivos para monitorizar os seus computadores portáteis durante os exames. Da mesma forma, plataformas como Discord e WhatsApp proíbem utilizadores que escrevem "self-bots" porque não utilizam a aplicação padrão.

Como podemos resolver isto?

A nossa mentalidade precisa de mudar, para que possa sempre executar um código em que confie. Se não quiser executar qualquer software apresentado, pode utilizá-lo ou encontrar (ou criar) uma alternativa você mesmo.

As organizações e empresas podem fornecer software de cliente que os seus utilizadores podem executar, mas a utilização desse software nunca deve ser obrigatória ou exigida. Deve ser sempre possível aos utilizadores escrever um cliente alternativo que interaja com um servidor da mesma forma.

Direitos semelhantes

Autonomia do software

Jardins murados limitam a liberdade. Qualquer pessoa deve ter o direito de reprogramar o seu dispositivo.
Abandonar a continuidade do software

Ao contrário do Rato Mickey, não poder utilizar software de uma pessoa morta pode prejudicar os seres humanos. Assim, o software deve tornar-se de domínio público no fim do seu ciclo de vida.
Garantia de software

O software não sofre erosão por causas naturais, pelo que merece uma esperança de vida mais longa.
