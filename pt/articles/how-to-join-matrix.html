Como se juntar à rede Matrix

Existem motivos humanitários bem como razões econômicas para se juntar à matriz. Este artigo pressupõe que você está disposto a tentar, e eu vou explicar a você como se juntar a ela!

A privacidade e a soberania de dados são garantidas na matriz, mas podem ser difíceis e confusas se juntar à rede. Este é um tutorial simples que ajudará você a configurar sua conta de matriz.

Guia rápido

Se você não pode escolher, faça as seguintes opções. Escolha elemento na etapa 1. Escolha nltrix.net Se você fala holandês e ao vivo na Europa, caso contrário, escolha matrix.org na etapa 2.

Passo 1: Escolha um cliente

Você tem que usar um aplicativo para falar na matriz. Não importa qual aplicativo você escolher, no entanto! Todos estão bem e todos podem se comunicar uns com os outros. Além disso, você sempre pode trocar aplicativos sempre que mudar de idéia.

Se você quiser escolher um cliente, você pode Encontre uma lista de clientes disponíveis aqui . Caso contrário, deixe-me dar algumas recomendações.

Elemento

elemento é o aplicativo que é ativamente mantido pela organização matricial.

Plataforma: Navegador, Windows, Linux, Android, iOS

  • ✅ Geralmente o aplicativo mais rápido para obter as atualizações mais recentes;
  • ✅ oficialmente suportado pela organização matricial;
  • ✅ Suporta toda a funcionalidade da matriz;
  • ❌ Feito para programadores e às vezes podem ser um pouco complicados.

Fluffychat.

Fluffychat. é um aplicativo de código aberto que é fácil de usar e muito fácil de usar. Sua funcionalidade é limitada, o que significa que é fácil entender, mas também limitado no que permite fazer na matriz. Este aplicativo é melhor adequado para crianças e para pessoas que não estão muito familiarizadas com a mídia social.

Plataforma: Navegador, Linux, Android, iOS

  • ✅ Fácil de usar e pegar;
  • ✅ Ui simples;
  • ❌ Limited. funcionalidade.

matrixcli.

Se você é um programador que odeia guis e gosta de executar comandos diretos no terminal para todos os seus comandos, então matrixcli é exatamente o que você precisa.

O que há para dizer? É um cliente de linha de comando. Aposto que os vim-amantes gostam desse tipo de cliente.

  • ✅ Super minimalista no desempenho do seu computador;
  • ❌ um enorme Não, para pessoas que não possam programar;
  • ❌ a falta de A interface pode torná-lo tedioso usar para algumas pessoas.

Miitrix.

O que?! Você pode falar com a rede Matrix do seu Nintendo 3DS?! Isso mesmo, com miitrix você pode! Eu não sei por que você gostaria, mas eu acho que você poderia.

  • ✅ Você pode usá-lo em um Nintendo 3DS;
  • ❌ Você só pode usá-lo em um Nintendo 3DS.

Passo 2: Escolha um homeserver

Você é livre para usar o servidor que quiser. Este servidor irá hospedar sua conta e seus dados. Atualmente, você Não é possível mover contas para diferentes casas de lares, para que você possa atualmente apenas switch entre homeservers por criando uma nova conta.

Eu recomendaria um dos seguintes caseiros. Alternativamente, você pode escolher um homeserver você mesmo. o Homeservers A partir desta lista deve estar bem.

Nome do servidor Domínio Descrição
Servidor de matriz Matrix.org.
  • Servidor oficial da matriz.
  • Garantia atualizada
  • Às vezes lagoso porque tem muitos usuários
Servidor nltrix. Onde está trix.net.
  • Servidor holandês gratuito de alto desempenho
  • Muito rápido para residentes na Holanda
Onde o servidor Vise .modular.im.
  • Alimentado por uma doação de uma comunidade
  • Realmente se preocupa com a privacidade

Etapa 3: criar uma conta

Instale o aplicativo que você escolheu na etapa 1. Certifique-se de escolher o seu homeserver direito antes você crie sua conta. Por exemplo, se você escolher o nltrix server na etapa 2, certifique-se de que você Crie uma conta no domínio nltrix.net . A maioria dos aplicativos tem matrix.org por padrão.
