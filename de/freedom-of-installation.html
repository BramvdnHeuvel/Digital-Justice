Bewahren Sie sich die Freiheit, zu installieren, was Sie wollen.

Sie allein entscheiden, was auf Ihrem Telefon, Laptop oder einem anderen Gerät installiert ist und was nicht. Wenn Sie ein elektronisches Gerät von einem Unternehmen kaufen, sollte es nicht in der Lage sein, Sie zu zwingen, bestimmte Anwendungen oder Software auf dem Gerät zu installieren.

Warum Freiheit der Installation?

Bei vielen Produkten wie einem Laptop oder einem Android-Smartphone ist es sinnvoll, dass Sie entscheiden, was auf Ihrem Computer läuft. Einige Smartphones enthalten jedoch Apps, die Sie nicht deinstallieren können - und die dennoch Arbeitsspeicher und Speicherplatz beanspruchen und manchmal sogar das gekaufte Gerät verlangsamen.

Während der Pandemie sind viele Universitäten dazu übergegangen, von den Studenten zu verlangen, dass sie eine Überwachungssoftware auf ihren Geräten installieren, um ihre Laptops während der Prüfungen zu überwachen. Auf die gleiche Weise sperren Plattformen wie Discord und WhatsApp Nutzer, die "Self-Bots" schreiben, weil sie nicht die Standard-App verwenden.

Wie können wir das Problem lösen?

Unsere Einstellung muss sich dahingehend ändern, dass Sie immer in der Lage sein sollten, Code auszuführen, dem Sie vertrauen. Wenn Sie eine vorgestellte Software nicht verwenden wollen, können Sie sie entweder benutzen oder selbst eine Alternative finden (oder erstellen).

Organisationen und Unternehmen können ihren Nutzern Client-Software zur Verfügung stellen, aber die Verwendung dieser Software sollte niemals verpflichtend oder vorgeschrieben sein. Es sollte den Nutzern immer möglich sein, einen alternativen Client zu schreiben, der auf die gleiche Weise mit dem Server interagiert.

Ähnliche Rechte

Software-Autonomie

Walled Gardens schränken die Freiheit ein. Jeder sollte das Recht haben, sein Gerät umzuprogrammieren.
Abandonware Continuity

Im Gegensatz zu Mickey Mouse kann es den Menschen schaden, wenn sie die Software einer toten Person nicht nutzen können. Daher sollte Software am Ende ihres Lebenszyklus gemeinfrei werden.
Software-Garantie

Software erodiert nicht durch natürliche Ursachen und verdient daher eine längere Lebenserwartung.
