Sie sind der einzige, der entscheidet, was auf Ihrem Telefon, Laptop oder einem anderen Gerät installiert ist oder nicht installiert ist. Wenn Sie ein elektronisches Gerät von einem Unternehmen kaufen, sollten Sie Sie nicht dazu zwingen können, bestimmte Apps oder Software zu haben, die darauf installiert sind.

Wie es funktioniert

Warum Installationsfreiheit ?

Für viele Produkte wie ein Laptop oder ein Android-Smartphone ist es sinnvoll, dass Sie entscheiden, was auf Ihrem Computer ausgeführt wird. Einige Smartphones enthalten jedoch Apps, die Sie nicht deinstallieren können - aber sie nehmen immer noch Speicher, Speicherplatz auf und verlangsamen sich manchmal sogar das Gerät, das Sie gekauft haben.

Während der Pandemie haben viele Universitäten begonnen, Studierende dazu zu erfordern, Proctoring-Software auf ihren Geräten zu installieren, um ihre Laptops während der Prüfungen zu überwachen. Ebenso, Plattformen wie Discord und WhatsApp Ban-Benutzer, die "Self-Bots" schreiben, weil sie die Standard-App nicht verwenden.

Wie können wir das beheben?

Unsere Denkweise muss ändern, dass Sie immer Code ausführen können, der Sie vertrauen. Wenn Sie keine präsentierte Software ausführen möchten, können Sie sie entweder verwenden oder (oder erstellen) eine Alternative selbst finden (oder erstellen).

Organisationen und Unternehmen können die Client-Software bereitstellen, die ihre Benutzer ausführen können, aber die Verwendung dieser Software sollte jedoch niemals verpflichtet oder erforderlich sein. Es sollte immer möglich sein, dass Benutzer einen alternativen Client schreiben, der mit einem Server auf dieselbe Weise interagiert.

Ähnliche Rechte

Software-Autonomie.

Mauerngärten begrenzen Freiheit. Jeder sollte das Recht haben, ihr Gerät neu zu programmieren.

Verlassene Kontinuität

Im Gegensatz zur Mickey-Maus kann keine Software von einem Toten von einem Toten verwenden. Daher sollte die Software am Ende des Lebenszyklus zu öffentlichen Domäne werden.

Software-Garantie.

Die Software erodiert nicht aus natürlichen Ursachen, verdient daher eine längere Lebenserwartung.
