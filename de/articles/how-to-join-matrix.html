So beitreten Sie dem Matrix-Netzwerk

Es gibt humanitäre Gründe sowie wirtschaftliche Gründe , um Matrix beizutreten. Dieser Artikel geht davon aus, dass Sie bereit sind, es einen Versuch zu geben, und ich werde Ihnen erklären, wie Sie mitzutreten!

Privatsphäre und Daten Souveränität sind auf Matrix garantiert, aber es kann schwierig sein und verwirrend sein, das Netzwerk anzuschließen. Dies ist ein einfaches Tutorial, mit dem Sie Ihr Matrix-Konto einrichten können.

Kurzanleitung

Wenn Sie nicht auswählen können, nehmen Sie die folgenden Optionen an. Wählen Sie in Schritt Element auf Schritt 1. Wählen Sie nltrux.net Wenn Sie Niederländisch sprechen und in Europa leben, wählen Sie ansonsten matrix.org auf Schritt 2.

Schritt 1: Wählen Sie einen Client aus

Sie müssen eine App verwenden, um auf der Matrix zu sprechen. Es spielt keine Rolle, welche App Sie wählen, jedoch! Alles ist in Ordnung, und alle können miteinander kommunizieren. Darüber hinaus können Sie immer Apps wechseln, wann immer Sie Ihre Meinung ändern.

Wenn Sie einen Kunden selbst auswählen möchten, können Sie eine Liste der verfügbaren Kunden hier finden. Ansonsten lassen Sie mich ein paar Empfehlungen geben.

Element

Element ist die App, die von der Matrixorganisation aktiv gewartet wird.

plattformen: Browser, Windows, Linux, Android, iOS

  • ✅ Normalerweise die schnellste App, um die neuesten Updates zu erhalten.
  • ✅ offiziell von der Matrixorganisation unterstützt;
  • ✅ Unterstützt alle Matrixfunktionen;
  • ❌ Hergestellt Programmierer und können manchmal etwas kompliziert sein.

Fluffychat.

fluffychat. ist eine Open Source-App, die einfach zu bedienen ist und sehr benutzerfreundlich ist. Seine Funktionalität ist begrenzt, was bedeutet, dass es sowohl leicht zu verstehen ist, sondern auch in dem, was Sie auf Matrix tun, begrenzt. Diese App passt am besten für Kinder und für Menschen, die sich nicht mit Social Media vertraut sind.

plattformen: Browser, Linux, Android, iOS

  • ✅ Einfach zu bedienen und abzuholen;
  • ✅ einfache ui;
  • ❌ Limited. Funktionalität.

MatrixCli.

Wenn Sie ein Programmierer sind, der GUIS hasst, und liebt es, direkte Befehle im Terminal für alle ihre Befehle auszuführen, dann ist matrixcli genau das du brauchst.

Was ist zu sagen? Es ist ein Befehlszeilen-Client. Ich wette, dass Vim-Liebhaber diese Art von Kunden genießen.

  • ✅ Super minimalistisch auf der Leistung Ihres Computers;
  • ❌ ein riesiges Nein, nein für Menschen, die nicht programmieren können;
  • ❌ das Fehlen von Die Schnittstelle kann es streichlosen, für manche Menschen zu verwenden.

Miantrix.

Was?! Sie können von Ihrem Nintendo 3DS mit dem Matrix-Netzwerk sprechen?! Das ist richtig, mit miitrix können Sie! Ich weiß nicht, warum du willst, aber ich denke du könntest.

  • ✅ Sie können es auf einem Nintendo-3DS verwenden.
  • ❌ Sie können es nur auf einem Nintendo-3DS verwenden.

Schritt 2: Wählen Sie einen Homeserver

Sie können den gewünschten Server kostenlos verwenden. Dieser Server hostet Ihr Konto und Ihre Daten. Derzeit, Sie Konten können nicht an verschiedene HomeServer eingestellt werden, sodass Sie derzeit nur Switch zwischen HomeVervers von ein neues Konto erstellen.

Ich würde eines der folgenden Homevers empfehlen. Alternativ können Sie einen Homeserver selbst auswählen. Die HomeStersvers aus dieser Liste sollte in Ordnung sein.

Servername Domain Beschreibung
Matrix-Server Matrix.org.
  • Offizieller Matrix-Server
  • Aktuelle Garantie
  • Manchmal laggy, weil es zu viele Benutzer hat
NltRix-Server Wo ist trix.net.
  • Hochleistungsfreier niederländischer Server
  • Sehr schnell für die Bewohner in den Niederlanden
Wo Server Rohr .modular.im.
  • Von einem Spenden von einer Gemeinschaft angetrieben
  • Kümmert sich wirklich um Privatsphäre

Schritt 3: Erstellen Sie ein Konto

Installieren Sie die App, die Sie in Schritt 1. Wählen Sie Ihren richtigen HomeServer vor aus Erstelle deinen Account. Wenn Sie beispielsweise den NLTRIX-Server in Schritt 2 auswählen, stellen Sie sicher, dass Sie das Erstellen Sie ein Konto auf der Domäne nltrux.net . Die meisten Apps haben standardmäßig matrix.org .

Schritt 4: Verbinden Sie sich mit Ihren Freunden

Wenn Sie Hilfe benötigen, beitreten Sie dem Raum #matrix: matrix.org . Hier helfen Fremde im Internet, Ihnen zu helfen, sich voranzutreiben. Fühlen Sie sich außerdem an mich an uns wenden Sie sich an mich an @bramvdnheuvel: nltrix.net für die Unterstützung.

Wenn ein Freund diesen Artikel an Sie geschickt hat, haben sie wahrscheinlich ein Konto und ich bin sicher, dass sie Ihnen gerne helfen werden. Senden Sie eine Chat-Anforderung an sie, und Sie haben Ihre eigene endgültige verschlüsselte Nachricht mit Ihrem Freund!
