Αρχική σελίδα Ψηφιακά δικαιώματα Άρθρα

Εξασφαλίζοντας ότι η εγγύηση λογισμικού διαρκεί πολύ περισσότερο

Σε αντίθεση με τα φυσικά αντικείμενα, το λογισμικό δεν φθείρεται με την πάροδο του χρόνου. Αν πληρώσετε για μια εφαρμογή smartphone, δεν θα πρέπει να χαλάσει μετά από λίγες ημέρες. Όπως κάθε άλλο προϊόν που αγοράζετε, έτσι και το λογισμικό θα πρέπει να έχει εγγύηση- μια εγγύηση που διαρκεί πολύ περισσότερο.

Πώς λειτουργεί

Γιατί εγγύηση λογισμικού;

Κάθε προϊόν που αγοράζεται στην Ευρωπαϊκή Ένωση έχει περιορισμένη εγγύηση: αν το προϊόν χαλάσει ή παρουσιάσει δυσλειτουργία κατά τη διάρκεια αυτής της περιόδου, μπορείτε είτε να το επισκευάσετε ή να το αντικαταστήσετε είτε να πάρετε πίσω τα χρήματά σας.

Η διάρκεια της εγγύησης εξαρτάται συνήθως από τον τύπο του προϊόντος. Είναι λογικό για ένα smartphone να διαρκεί τουλάχιστον 2 χρόνια, αλλά μια μπανάνα μάλλον δεν θα διαρκέσει τόσο πολύ. Για το λογισμικό, ωστόσο, δεν υπάρχει καθορισμένη διάρκεια για το πόσο θα πρέπει να διαρκεί "λογικά" το λογισμικό, αν το αγοράσετε.

Πώς μπορούμε να το διορθώσουμε αυτό;

Αν αγοράσετε μια εφαρμογή από ένα κατάστημα εφαρμογών, θα πρέπει να λειτουργεί - και αν δεν λειτουργεί ή αν γίνει άχρηστη μετά από λίγο, θα πρέπει να πάρετε πίσω τα χρήματά σας.

Θα πρέπει να αρχίσουμε να αντιμετωπίζουμε τις εφαρμογές και το λογισμικό σαν προϊόντα: αν το πληρώνεις, θα πρέπει να υπάρχει εγγύηση. Οι ειδικοί θα πρέπει να καθορίσουν πόσο καιρό θα πρέπει λογικά να λειτουργεί μια εφαρμογή, και στη συνέχεια αυτές οι εγγυήσεις θα πρέπει να νομοθετηθούν.

Προσωπική εμπειρία

Κανείς δεν ξέρει

Πόσο διαρκεί η εγγύηση για το λογισμικό; 🔖 Αποδεικνύεται ότι αυτό είναι κάτι που κανείς δεν γνωρίζει.

Τηλεφώνησα στην ολλανδική αρχή για τις εγγυήσεις, στην τεχνική αρχή, σε έναν πωλητή λογισμικού και στο νομικό γκισέ για τους πολίτες, και κανένας από αυτούς δεν ήταν σε θέση να μου δώσει απάντηση. 🙈

Με συμβούλευσαν να συμβουλευτώ έναν δικηγόρο γιατί "ίσως εκείνος να ξέρει".

Παρόμοια δικαιώματα

Δικαίωμα στην επισκευή

Θα πρέπει να μπορείτε να επισκευάζετε χαλασμένα ηλεκτρονικά εξαρτήματα αντί να χρειάζεται να αγοράσετε μια νέα συσκευή.
Συνέχεια της εγκατάλειψης του λογισμικού

Σε αντίθεση με τον Μίκυ Μάους, το να μην μπορείτε να χρησιμοποιήσετε λογισμικό από ένα νεκρό άτομο μπορεί να βλάψει τους ανθρώπους. Ως εκ τούτου, το λογισμικό θα πρέπει να καθίσταται κοινό κτήμα στο τέλος του κύκλου ζωής του.
Ελευθερία εγκατάστασης
