Εξασφαλίζοντας ότι η εγγύηση του λογισμικού διαρκεί πολύ περισσότερο

Σε αντίθεση με τα φυσικά αντικείμενα, το λογισμικό δεν διαβρώνεται με την πάροδο του χρόνου. Εάν πληρώσετε για μια εφαρμογή smartphone, δεν πρέπει να σπάει μετά από λίγες μέρες. Ακριβώς όπως οποιοδήποτε άλλο προϊόν που αγοράζετε, το λογισμικό θα πρέπει να έχει εγγύηση. ένα που διαρκεί πολύ περισσότερο.

Πως δουλεύει

Γιατί Εγγύηση λογισμικού ;

Οποιοδήποτε προϊόν που αγοράστηκε στην Ευρωπαϊκή Ένωση έχει περιορισμένη εγγύηση: εάν το προϊόν σπάσει ή κακή συμπεριφορά κατά τη διάρκεια αυτής της περιόδου, μπορείτε είτε να λάβετε το προϊόν επισκευασμένο είτε να αντικατασταθεί, ή μπορείτε να πάρετε τα χρήματά σας πίσω.

Η διάρκεια της εγγύησης εξαρτάται συνήθως από τον τύπο του προϊόντος. Είναι λογικό για ένα smartphone να διαρκέσει τουλάχιστον 2 χρόνια, αλλά μια μπανάνα πιθανότατα δεν θα διαρκέσει τόσο πολύ. Για το Software , ωστόσο, δεν υπάρχει καθορισμένη διάρκεια του χρόνου λογισμικού θα πρέπει να "λογικά" να διαρκέσει αν το αγοράσετε.

Πώς μπορούμε να διορθώσουμε αυτό;

Εάν αγοράσετε μια εφαρμογή από ένα κατάστημα εφαρμογών, θα πρέπει να λειτουργεί - και αν δεν λειτουργεί ή αν γίνει ασυνήθιστο μετά από λίγο, θα πρέπει να πάρετε τα χρήματά σας πίσω.

Πρέπει να ξεκινήσουμε τη θεραπεία εφαρμογών και λογισμικού όπως τα προϊόντα: αν το πληρώσετε, θα πρέπει να υπάρχει εγγύηση. Οι ειδικοί θα πρέπει να καθορίσουν πόσο καιρό θα πρέπει να εργαστεί μια εφαρμογή, και στη συνέχεια πρέπει να νομοθετούν αυτές οι εγγυήσεις.
