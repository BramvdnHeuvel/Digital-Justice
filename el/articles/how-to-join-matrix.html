Πώς να ενταχθείτε στο δίκτυο Matrix

Υπάρχουν ανθρωπιστικοί λόγοι καθώς και οικονομικοί λόγοι για να συμμετάσχετε στη Matrix. Αυτό το άρθρο προϋποθέτει ότι είστε πρόθυμοι να το δοκιμάσετε και θα σας εξηγήσω πώς να ενταχθείτε σε αυτό!

Η ιδιωτικότητα και η κυριαρχία των δεδομένων είναι εγγυημένες στο Matrix, αλλά μπορεί να είναι δύσκολη και συγκεχυμένη η ένταξη στο δίκτυο. Αυτό είναι ένα απλό σεμινάριο που θα σας βοηθήσει να δημιουργήσετε το λογαριασμό σας στο Matrix.

Γρήγορος οδηγός

Εάν δεν μπορείτε να επιλέξετε, επιλέξτε τις ακόλουθες επιλογές. Επιλέξτε το Element στο βήμα 1. Επιλέξτε nltrix.net αν μιλάτε ολλανδικά και ζείτε στην Ευρώπη, διαφορετικά επιλέξτε matrix.org στο βήμα 2.

Βήμα 1: επιλέξτε έναν πελάτη

Πρέπει να χρησιμοποιήσετε μια εφαρμογή για να μιλήσετε στο Matrix. Ωστόσο, δεν έχει σημασία ποια εφαρμογή θα επιλέξετε! Όλες είναι καλές και όλες μπορούν να επικοινωνήσουν μεταξύ τους. Επιπλέον, μπορείτε πάντα να αλλάζετε εφαρμογές όποτε αλλάζετε γνώμη.

Αν θέλετε να επιλέξετε μόνοι σας έναν πελάτη, μπορείτε να βρείτε μια λίστα με τους διαθέσιμους πελάτες εδώ. Διαφορετικά, επιτρέψτε μου να σας δώσω μερικές συστάσεις.

Στοιχείο

Element είναι η εφαρμογή που συντηρείται ενεργά από τον οργανισμό Matrix.

Πλατφόρμες: Browser, Windows, Linux, Android, iOS

  • ✅ Συνήθως η πιο γρήγορη εφαρμογή για να λάβετε τις τελευταίες ενημερώσεις,
  • ✅ Υποστηρίζεται επίσημα από τον οργανισμό Matrix,
  • ✅ Υποστηρίζει όλες τις λειτουργίες του Matrix,
  • ❌ Φτιαγμένο για προγραμματιστές και μπορεί μερικές φορές να είναι λίγο περίπλοκο.

FluffyChat

Το FluffyChat είναι μια εφαρμογή ανοιχτού κώδικα που είναι εύχρηστη και πολύ φιλική προς το χρήστη. Η λειτουργικότητά του είναι περιορισμένη, πράγμα που σημαίνει ότι είναι και εύκολο να κατανοηθεί αλλά και περιορισμένο σε ό,τι σας επιτρέπει να κάνετε στο Matrix. Αυτή η εφαρμογή είναι καταλληλότερη για παιδιά και για άτομα που δεν είναι πολύ εξοικειωμένα με τα μέσα κοινωνικής δικτύωσης.

Πλατφόρμες: Browser, Linux, Android, iOS

  • ✅ Εύκολο στη χρήση και την παραλαβή,
  • ✅ Απλό UI,
  • ❌ Περιορισμένη λειτουργικότητα.

matrixcli

Αν είστε προγραμματιστής που μισεί τα γραφικά περιβάλλοντα και λατρεύει να εκτελεί απευθείας εντολές στο τερματικό για όλες τις εντολές του, τότε το matrixcli είναι ακριβώς αυτό που χρειάζεστε.

Τι υπάρχει να πούμε; Είναι ένας πελάτης γραμμής εντολών. Στοιχηματίζω ότι οι λάτρεις του Vim απολαμβάνουν αυτού του είδους τον πελάτη.

  • ✅ Σούπερ μινιμαλιστικό για τις επιδόσεις του υπολογιστή σας,
  • ❌ Ένα τεράστιο όχι για άτομα που δεν μπορούν να προγραμματίσουν,
  • ❌ Η έλλειψη διεπαφής μπορεί να κάνει τη χρήση της κουραστική για ορισμένους ανθρώπους.

Βήμα 2: επιλέξτε έναν οικιακό διακομιστή

Είστε ελεύθεροι να χρησιμοποιήσετε όποιον διακομιστή θέλετε. Αυτός ο διακομιστής θα φιλοξενεί το λογαριασμό σας και τα δεδομένα σας. Προς το παρόν, δεν μπορείτε να μετακινήσετε λογαριασμούς σε διαφορετικούς homeervers, οπότε μπορείτε προς το παρόν να αλλάξετε μεταξύ homeervers μόνο δημιουργώντας έναν νέο λογαριασμό.

Θα πρότεινα έναν από τους ακόλουθους homeervers. Εναλλακτικά, μπορείτε να επιλέξετε έναν homeerver μόνοι σας. Οι homeservers από αυτή τη λίστα θα πρέπει να είναι μια χαρά.

Όνομα διακομιστή Τομέας Περιγραφή
Matrix Server matrix.org
  • Επίσημος διακομιστής Matrix
  • Ενημερωμένη εγγύηση
  • Μερικές φορές καθυστερεί επειδή έχει πολλούς χρήστες
NLtrix Server nltrix.net
  • Δωρεάν ολλανδικός διακομιστής υψηλής απόδοσης
  • Πολύ γρήγορα για τους κατοίκους των Κάτω Χωρών
Διακομιστής KDE kde.modular.im
  • Powered by a donations from a community
  • Πραγματικά ενδιαφέρεται για την ιδιωτική ζωή.

Βήμα 3: δημιουργία λογαριασμού

Εγκαταστήστε την εφαρμογή που επιλέξατε στο βήμα 1. Βεβαιωθείτε ότι έχετε επιλέξει το σωστό home server πριν δημιουργήσετε το λογαριασμό σας. Για παράδειγμα, αν επιλέξετε τον NLtrix server στο βήμα 2, βεβαιωθείτε ότι έχετε δημιουργήσει έναν λογαριασμό στο domain nltrix.net. Οι περισσότερες εφαρμογές έχουν από προεπιλογή το matrix.org.

Βήμα 4: συνδεθείτε με τους φίλους σας

Αν χρειάζεστε βοήθεια, μπείτε στην αίθουσα #matrix:matrix.org. Εδώ, άγνωστοι στο διαδίκτυο είναι πρόθυμοι να σας βοηθήσουν να προχωρήσετε. Επιπλέον, μη διστάσετε να επικοινωνήσετε μαζί μου στο @bramvdnheuvel:nltrix.net για υποστήριξη.

Αν κάποιος φίλος σας έστειλε αυτό το άρθρο, πιθανότατα θα έχει λογαριασμό και είμαι σίγουρος ότι θα χαρεί να σας βοηθήσει. Στείλτε τους ένα αίτημα συνομιλίας και θα έχετε το δικό σας πλήρως κρυπτογραφημένο από άκρο σε άκρο μήνυμα με τον φίλο σας!

Βήμα 5: συνδεθείτε με άτομα σε άλλες πλατφόρμες

Το όλο νόημα του Matrix είναι ότι μπορεί να μιλήσει ακόμα και με ανθρώπους σε άλλες πλατφόρμες! Σύντομα θα γράψω άρθρα για το πώς μπορείτε να συνδέσετε τις συνομιλίες σας στο Telegram και στο Discord. Μόλις τα γράψω, θα τα δημοσιεύσω εδώ κάτω.

Μπορείτε επίσης να το διαπιστώσετε μόνοι σας στο t2bot.io.

Προς το παρόν, απολαύστε τις πραγματικά ιδιωτικές σας συζητήσεις στο Matrix!
