Πώς να συμμετάσχετε στο δίκτυο Matrix

Υπάρχουν ανθρωπιστικοί λόγοι καθώς και Οικονομικοί λόγοι για να συμμετάσχετε στη μήτρα. Αυτό το άρθρο υποθέτει ότι είστε διατεθειμένοι να το δοκιμάσετε, και θα σας εξηγήσω πώς να συμμετάσχετε!

Το ιδιωτικό απόρρητο και η κυριαρχία δεδομένων εγγυάται στη Matrix, αλλά μπορεί να είναι δύσκολη και συγκεχυμένη να ενταχθούν στο δίκτυο. Αυτό είναι ένα απλό φροντιστήριο που θα σας βοηθήσει να ρυθμίσετε το λογαριασμό σας Matrix.

Γρήγορος οδηγός

Εάν δεν μπορείτε να επιλέξετε, λάβετε τις ακόλουθες επιλογές. Επιλέξτε στοιχείο στο βήμα 1. επιλέξτε nltrix.net αν μιλάτε ολλανδικά και ζείτε στην Ευρώπη, διαφορετικά επιλέξτε matrix.org στο βήμα 2.

Βήμα 1: Επιλέξτε έναν πελάτη

Πρέπει να χρησιμοποιήσετε μια εφαρμογή για να μιλήσετε στο Matrix. Δεν έχει σημασία ποια εφαρμογή επιλέγετε, ωστόσο! Όλα είναι καλά, και όλοι μπορούν να επικοινωνούν μεταξύ τους. Επιπλέον, μπορείτε πάντα να αλλάζετε εφαρμογές κάθε φορά που αλλάζετε το μυαλό σας.

Αν θέλετε να επιλέξετε έναν πελάτη μόνοι σας, μπορείτε να επιτρέψετε βρείτε μια λίστα διαθέσιμων πελατών εδώ . Διαφορετικά, επιτρέψτε μου να σας δώσω κάποιες συστάσεις.

Στοιχείο

ΣΤΟΙΧΕΙΟ είναι η εφαρμογή που διατηρεί ενεργά από τον οργανισμό Matrix.

Πλατφόρμες: Browser, Windows, Linux, Android, iOS

  • ✅ Συνήθως η ταχύτερη εφαρμογή για να λάβετε τις τελευταίες ενημερώσεις.
  • ✅ επίσημα υποστηρίζεται από την οργάνωση Matrix.
  • ✅ Υποστηρίζει όλες τις λειτουργίες μήτρας.
  • ❌ έγινε για προγραμματιστές και μπορεί μερικές φορές να είναι λίγο περίπλοκο.

Χνούδι

χνουδιά είναι μια εφαρμογή ανοιχτού κώδικα που είναι εύκολη στη χρήση και πολύ φιλική προς το χρήστη. Η λειτουργικότητά του είναι περιορισμένη, πράγμα που σημαίνει ότι είναι τόσο εύκολο να κατανοηθεί αλλά και περιορισμένος σε αυτό που σας επιτρέπει να κάνετε στο Matrix. Αυτή η εφαρμογή είναι καλύτερη εφαρμογή για παιδιά και για ανθρώπους που δεν είναι πολύ εξοικειωμένοι με τα κοινωνικά μέσα.

Πλατφόρμες: Browser, Linux, Android, iOS

  • ✅ Εύκολο στη χρήση και σηκώστε.
  • ✅ Απλό UI;
  • ❌ Limited λειτουργικότητα.

Βήμα 2: Επιλέξτε ένα homeserver

Είστε ελεύθεροι να χρησιμοποιήσετε όποιο διακομιστή που σας αρέσει. Αυτός ο διακομιστής θα φιλοξενήσει το λογαριασμό σας και τα δεδομένα σας. Επί του παρόντος, εσείς Δεν είναι δυνατή η μετακίνηση λογαριασμών σε διαφορετικούς homeservers, έτσι μπορείτε να μετατρέψετε αυτήν τη στιγμή μόνο switch μεταξύ Homeservers από δημιουργώντας ένα νέο λογαριασμό.

Θα συνιστούσα έναν από τους ακόλουθους homeservers. Εναλλακτικά, μπορείτε να επιλέξετε ένα homeserver μόνοι σας. ο Οι homeservers από αυτόν τον κατάλογο θα πρέπει να είναι ωραία.

Βήμα 3: Δημιουργία λογαριασμού

Εγκαταστήστε την εφαρμογή που επιλέξατε στο βήμα 1. Βεβαιωθείτε ότι έχετε επιλέξει το δεξί σας εξοχικό σπίτι πριν από you Δημιούργησε τον λογαριασμό σου. Για παράδειγμα, αν επιλέξετε το Nltrix Server στο βήμα 2, βεβαιωθείτε ότι έχετε Δημιουργήστε ένα λογαριασμό στον τομέα nltrix.net . Οι περισσότερες εφαρμογές έχουν matrix.org από προεπιλογή.

Βήμα 4: Συνδεθείτε στους φίλους σας

Εάν χρειάζεστε βοήθεια, συμμετέχετε στο δωμάτιο #matrix: matrix.org . Εδώ, οι ξένοι στο Διαδίκτυο είναι στην ευχάριστη θέση να σας βοηθήσουν να προχωρήσετε. Επιπλέον, μπορείτε να επικοινωνήσετε μαζί μου στο @ crramvdnheuvel: nltrix.net για υποστήριξη.

Εάν ένας φίλος έστειλε αυτό το άρθρο σε εσάς, πιθανότατα θα έχουν λογαριασμό και είμαι σίγουρος ότι θα χαρούμε να σας βοηθήσουν. Στείλτε ένα αίτημα συνομιλίας σε αυτά και θα έχετε το δικό σας πλήρως το τελικό κρυπτογραφημένο μήνυμα με τον φίλο σας!
