Διατηρήστε την ελευθερία να εγκαταστήσετε ό,τι θέλετε

Εσείς είστε ο μόνος που μπορεί να αποφασίσει τι είναι ή δεν είναι εγκατεστημένο στο τηλέφωνο, το φορητό υπολογιστή ή άλλη συσκευή σας. Αν αγοράζετε μια ηλεκτρονική συσκευή από μια εταιρεία, δεν θα πρέπει να μπορεί να σας αναγκάσει να έχετε συγκεκριμένες εφαρμογές ή λογισμικό εγκατεστημένο σε αυτήν.

Πώς λειτουργεί

Γιατί ελευθερία εγκατάστασης;

Για πολλά προϊόντα, όπως ένας φορητός υπολογιστής ή ένα smartphone Android, είναι λογικό να αποφασίζετε εσείς τι τρέχει στον υπολογιστή σας. Ωστόσο, ορισμένα smartphones περιέχουν εφαρμογές που δεν μπορείτε να απεγκαταστήσετε - αλλά εξακολουθούν να καταλαμβάνουν μνήμη, αποθηκευτικό χώρο και μερικές φορές ακόμη και να επιβραδύνουν τη συσκευή που αγοράσατε.

Κατά τη διάρκεια της πανδημίας, πολλά πανεπιστήμια άρχισαν να απαιτούν από τους φοιτητές να εγκαθιστούν λογισμικό επιτηρητών στις συσκευές τους για να παρακολουθούν τους φορητούς υπολογιστές τους κατά τη διάρκεια των εξετάσεων. Με τον ίδιο τρόπο, πλατφόρμες όπως το Discord και το WhatsApp απαγορεύουν τους χρήστες που γράφουν "self-bots" επειδή δεν χρησιμοποιούν την τυπική εφαρμογή.

Πώς μπορούμε να το διορθώσουμε αυτό;

Η νοοτροπία μας πρέπει να αλλάξει, ότι θα πρέπει πάντα να μπορείτε να εκτελείτε κώδικα που εμπιστεύεστε. Αν δεν θέλετε να εκτελέσετε κάποιο παρουσιαζόμενο λογισμικό, μπορείτε είτε να το χρησιμοποιήσετε είτε να βρείτε (ή να δημιουργήσετε) μια εναλλακτική λύση μόνοι σας.

Οι οργανισμοί και οι εταιρείες μπορούν να παρέχουν λογισμικό πελάτη που οι χρήστες τους μπορούν να εκτελούν, αλλά η χρήση αυτού του λογισμικού δεν θα πρέπει ποτέ να είναι υποχρεωτική ή να απαιτείται. Θα πρέπει πάντα να είναι δυνατό για τους χρήστες να γράφουν ένα εναλλακτικό πρόγραμμα-πελάτη που να αλληλεπιδρά με έναν διακομιστή με τον ίδιο τρόπο.

Παρόμοια δικαιώματα

Αυτονομία λογισμικού

Οι περιφραγμένοι κήποι περιορίζουν την ελευθερία. Ο καθένας θα πρέπει να έχει το δικαίωμα να επαναπρογραμματίσει τη συσκευή του.
Συνέχεια της εγκατάλειψης του λογισμικού

Σε αντίθεση με τον Μίκυ Μάους, το να μην μπορείτε να χρησιμοποιήσετε λογισμικό από ένα νεκρό άτομο μπορεί να βλάψει τους ανθρώπους. Ως εκ τούτου, το λογισμικό θα πρέπει να καθίσταται κοινό κτήμα στο τέλος του κύκλου ζωής του.
Εγγύηση λογισμικού

Το λογισμικό δεν διαβρώνεται από φυσικά αίτια, επομένως αξίζει μεγαλύτερο προσδόκιμο ζωής.
