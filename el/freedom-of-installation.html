Είστε ο μόνος που αποφασίζετε τι είναι ή δεν είναι εγκατεστημένο στο τηλέφωνο, φορητό σας ή σε άλλη συσκευή. Εάν αγοράσετε μια ηλεκτρονική συσκευή από μια εταιρεία, δεν θα πρέπει να είναι σε θέση να σας αναγκά να έχετε συγκεκριμένες εφαρμογές ή λογισμικό εγκατεστημένο σε αυτό.

Πως δουλεύει

Γιατί Ελευθερία εγκατάστασης ;

Για πολλά προϊόντα όπως ένα φορητό υπολογιστή ή ένα smartphone Android, έχει νόημα ότι αποφασίζετε τι τρέχει στον υπολογιστή σας. Ωστόσο, ορισμένα smartphones περιέχουν εφαρμογές που δεν μπορείτε να απεγκαταστήσετε - αλλά εξακολουθούν να αναλαμβάνουν μνήμη, αποθηκευτικό χώρο και μερικές φορές να επιβραδύνουν τη συσκευή που αγοράσατε.

Κατά τη διάρκεια της πανδημίας, πολλά πανεπιστήμια έχουν αρχίσει να απαιτούν από τους μαθητές να εγκαταστήσουν το λογισμικό Proctoring στις συσκευές τους για να παρακολουθήσουν τους φορητούς υπολογιστές τους κατά τη διάρκεια εξετάσεων. Με τον ίδιο τρόπο, οι πλατφόρμες όπως η Discord και Whatsapp απαγορεύει τους χρήστες που γράφουν "αυτο-bots" επειδή δεν χρησιμοποιούν την τυπική εφαρμογή.

Πώς μπορούμε να διορθώσουμε αυτό;

Η νοοτροπία μας πρέπει να αλλάξει ότι πρέπει πάντα να είστε σε θέση να εκτελέσετε κώδικα που εμπιστεύεστε. Εάν δεν θέλετε να εκτελέσετε οποιοδήποτε παρουσιασμένο λογισμικό, μπορείτε είτε να το χρησιμοποιήσετε είτε να βρείτε (ή να δημιουργήσετε) μια εναλλακτική λύση.

Οι οργανισμοί και οι εταιρείες μπορούν να παραδώσουν λογισμικό πελάτη που μπορούν να τρέξουν οι χρήστες τους, αλλά η χρήση αυτού του λογισμικού δεν πρέπει ποτέ να υποχρεωθεί ή να απαιτείται. Θα πρέπει πάντα να είναι δυνατό για τους χρήστες να γράψουν έναν εναλλακτικό πελάτη που αλληλεπιδρά με ένα διακομιστή με τον ίδιο τρόπο.
