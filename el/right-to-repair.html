Αρχική σελίδα Ψηφιακά δικαιώματα Άρθρα

Ας πάρουμε πίσω το δικαίωμά μας να επισκευάζουμε οτιδήποτε χαλάει

Τα περισσότερα ηλεκτρονικά προϊόντα φθείρονται με την πάροδο του χρόνου: οι λαστιχένιες λαβές μπορεί να φθαρούν, ένας ανεμιστήρας σε έναν υπολογιστή μπορεί να σταματήσει να περιστρέφεται ή ο φορτιστής σας μπορεί να σπάσει. Σε αυτές τις περιπτώσεις, θα πρέπει να μπορείτε να επισκευάσετε τα σπασμένα μέρη αντί να χρειαστεί να αγοράσετε μια εντελώς νέα συσκευή.

Πώς λειτουργεί

Γιατί το Right to Repair;

Όταν το smartphone σας χαλάει επειδή ένα μικρό τσιπ ή ηλεκτρονικό εξάρτημα έχει σταματήσει να λειτουργεί, ο αρχικός κατασκευαστής συχνά επισκευάζει τη συσκευή μόνο σε τόσο υψηλή τιμή που μπορείτε κάλλιστα να αγοράσετε ένα νέο τηλέφωνο. Η επισκευή μπορεί συχνά να διαρκέσει εβδομάδες και όλα τα δεδομένα διαγράφονται από τη συσκευή κατά τη διάρκεια της επισκευής, γεγονός που σας αφήνει έναν μεγάλο λογαριασμό και την απώλεια όλων των δεδομένων σας στο smartphone σας.

Στην πραγματικότητα, πολλά μεμονωμένα τσιπ κοστίζουν μόνο ένα μικρό ποσό, και η επισκευή δεν θα ήταν τόσο ακριβή αν οι κατασκευαστές είχαν συνεργεία επισκευής ως ανταγωνισμό. Αυτό όμως δεν συμβαίνει, επειδή οι εταιρείες βάζουν ρητά τα εργοστάσια να αρνούνται να πουλήσουν ηλεκτρονικά εξαρτήματα σε συνεργεία επισκευών. Αυτό είναι ένα ουσιαστικό κλείσιμο της αγοράς επισκευών και επιτρέπει στους προμηθευτές να χρεώνουν τις επισκευές σε γελοίες τιμές.

Πώς μπορούμε να το διορθώσουμε αυτό;

Δεν θα έπρεπε να χρειάζεται να αγοράζετε κάθε χρόνο έναν καινούργιο και ακριβό εκτυπωτή. Αντ' αυτού, θα πρέπει να μπορείτε να επισκευάζετε τον εκτυπωτή σας έναντι μικρού αντιτίμου, αν χαλάσει κάτι μικρό. Οι κατασκευαστές ηλεκτρονικών συσκευών θα πρέπει να είναι υποχρεωμένοι να μοιράζονται όλες τις πληροφορίες που απαιτούνται για την επισκευή της συσκευής και όλα τα εξαρτήματα θα πρέπει να είναι προσβάσιμα για να τα αγοράσει ο καθένας.

Το βίντεο είναι στα αγγλικά, αλλά υπάρχουν διαθέσιμοι υπότιτλοι.

Δεν μπορείτε να επισκευάσετε το iPhone 13 σας

Όπως αναφέρει η εκστρατεία Right to Repair Europe, το iPhone 13 απενεργοποιεί το Face ID αν επισκευάσετε τη σπασμένη οθόνη σας, ακόμη και αν επισκευάζεται με επίσημο ανταλλακτικό της Apple.

Η αδυναμία επισκευής των iPhone για τα ανεξάρτητα συνεργεία δίνει στην Apple μεγαλύτερη ελευθερία να απαιτεί εξωφρενικές τιμές για τις επισκευές τηλεφώνων. Μια επισκευασμένη οθόνη δεν αποτελεί κίνδυνο για την ασφάλεια και δεν είναι λογικό να μπλοκάρονται λειτουργίες του τηλεφώνου όπως το Face ID ως μέτρο ασφαλείας.

Παρόμοια δικαιώματα

Διαλειτουργικότητα

Η επικοινωνία δεν πρέπει να περιορίζεται στην πλατφόρμα που χρησιμοποιείτε. Ο καθένας δικαιούται να επικοινωνεί με οποιονδήποτε, ανεξάρτητα από την εφαρμογή που χρησιμοποιεί.
Αυτονομία λογισμικού

Οι περιφραγμένοι κήποι περιορίζουν την ελευθερία. Ο καθένας θα πρέπει να έχει το δικαίωμα να επαναπρογραμματίσει τη συσκευή του.
Εγγύηση λογισμικού

Το λογισμικό δεν διαβρώνεται από φυσικά αίτια, επομένως αξίζει μεγαλύτερο προσδόκιμο ζωής.
