Τα περισσότερα ηλεκτρονικά προϊόντα διαβρώνονται με την πάροδο του χρόνου: οι λαβές από καουτσούκ μπορούν να φθείρονται, ένας ανεμιστήρας σε έναν υπολογιστή ενδέχεται να σταματήσει την περιστροφή ή ο φορτιστής σας μπορεί να σπάσει. Σε αυτές τις περιπτώσεις, θα πρέπει να είστε σε θέση να επισκευάσετε τα σπασμένα μέρη αντί να αγοράσετε μια πλήρη νέα συσκευή.

Πως δουλεύει

Γιατί Δικαίωμα επισκευής ;

Όταν το smartphone σας καταρρεύσει επειδή ένα μικρό τσιπ ή ηλεκτρονικό τμήμα έχει σταματήσει να λειτουργεί, ο αρχικός κατασκευαστής επισκευάζει συχνά τη συσκευή μόνο σε μια τιμή τόσο υψηλή που μπορείτε επίσης να αγοράσετε ένα νέο τηλέφωνο. Η επισκευή μπορεί συχνά να διαρκέσει εβδομάδες και όλα τα δεδομένα σκουπίζονται από τη συσκευή κατά τη διάρκεια της επισκευής, το οποίο σας αφήνει ένα μεγάλο νομοσχέδιο και απώλεια όλων των δεδομένων σας στο smartphone σας.

Στην πραγματικότητα, πολλές μεμονωμένες μάρκες κοστίζουν μόνο ένα μικρό ποσό και η επισκευή δεν θα ήταν τόσο δαπανηρή εάν οι κατασκευαστές είχαν καταστήματα επισκευής ως ανταγωνισμός. Αυτό δεν συμβαίνει, ωστόσο, επειδή οι εταιρείες λαμβάνουν ρητά εργοστάσια για να αρνηθούν την πώληση ηλεκτρονικών εξαρτημάτων για την επισκευή καταστημάτων. Πρόκειται για ένα αποτελεσματικό κλείσιμο της αγοράς επισκευής και επιτρέπει στους πωλητές να χρεώνουν επισκευές σε γελοίες τιμές.

Πώς μπορούμε να διορθώσουμε αυτό;

Δεν πρέπει να χρειάζεται να αγοράσετε ένα νέο και ακριβό εκτυπωτή κάθε χρόνο. Αντ 'αυτού, θα πρέπει να είστε σε θέση να επισκευάσετε τον εκτυπωτή σας για μια μικρή τιμή αν κάτι μικρό σπάσιμο. Οι κατασκευαστές ηλεκτρονικών συσκευών πρέπει να αναγκάζονται να μοιράζονται όλες τις πληροφορίες που απαιτούνται για την επαναφορά της συσκευής και όλα τα μέρη θα πρέπει να είναι προσβάσιμα για να αγοράσουν για οποιονδήποτε.
