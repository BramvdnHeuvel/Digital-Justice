Comment rejoindre le réseau Matrix

Il existe des raisons humanitaires ainsi que des raisons économiques de rejoindre Matrix. Cet article part du principe que vous êtes prêt à essayer, et je vais vous expliquer comment la rejoindre !

La confidentialité et la souveraineté des données sont garanties sur Matrix, mais il peut être difficile et déroutant de rejoindre le réseau. Voici un tutoriel simple qui vous aidera à configurer votre compte Matrix.

Guide rapide

Si vous ne pouvez pas choisir, prenez les options suivantes. Choisissez Element à l'étape 1. Choisissez nltrix.net si vous parlez néerlandais et vivez en Europe, sinon choisissez matrix.org à l'étape 2.

Étape 1 : choisir un client

Vous devez utiliser une application pour parler sur le Matrix. Le choix de l'application n'a cependant pas d'importance ! Toutes conviennent et toutes peuvent communiquer entre elles. De plus, vous pouvez toujours changer d'application si vous changez d'avis.

Si vous voulez choisir un client vous-même, vous pouvez trouver une liste des clients disponibles ici. Sinon, laissez-moi vous donner quelques recommandations.

Élément

Element est l'application qui est activement maintenue par l'organisation Matrix.

Platforms: Navigateur, Windows, Linux, Android, iOS

  • ✅ Habituellement, l'application la plus rapide pour obtenir les dernières mises à jour ;
  • ✅ Officiellement soutenu par l'organisation Matrix ;
  • ✅ Supporte toutes les fonctionnalités de Matrix ;
  • ❌ Fait pour les programmeurs et peut parfois être un peu compliqué.

FluffyChat

FluffyChat est une application open source facile à utiliser et très conviviale. Ses fonctionnalités sont limitées, ce qui signifie qu'elle est à la fois facile à comprendre mais aussi limitée dans ce qu'elle vous permet de faire sur Matrix. Cette application est plus adaptée aux enfants et aux personnes qui ne sont pas très familières avec les médias sociaux.

Platforms: Navigateur, Linux, Android, iOS

  • ✅ Facile à utiliser et à prendre en main ;
  • ✅ Interface utilisateur simple ;
  • ❌ Fonctionnalité limitée.

matrixcli

Si vous êtes un programmeur qui déteste les interfaces graphiques et qui aime exécuter des commandes directes dans le terminal pour toutes ses commandes, alors matrixcli est exactement ce dont vous avez besoin.

Qu'y a-t-il à dire ? C'est un client en ligne de commande. Je parie que les amoureux de Vim apprécient ce type de client.

  • ✅ Super minimaliste sur les performances de votre ordinateur ;
  • ❌ Un énorme non-non pour les gens qui ne savent pas programmer ;
  • ❌ L'absence d'interface peut rendre son utilisation fastidieuse pour certaines personnes.

Miitrix

Quoi ? ! Vous pouvez parler au réseau Matrix depuis votre Nintendo 3DS ? ! C'est exact, avec Miitrix vous le pouvez ! Je ne sais pas pourquoi vous voudriez le faire, mais je suppose que vous pourriez.

  • ✅ Vous pouvez l'utiliser sur une Nintendo 3DS ;
  • ❌ Vous ne pouvez l'utiliser que sur une Nintendo 3DS.

Étape 2 : choisir un serveur domestique

Vous êtes libre d'utiliser le serveur de votre choix. Ce serveur hébergera votre compte et vos données. Actuellement, vous ne pouvez pas déplacer les comptes vers différents serveurs domestiques, donc vous ne pouvez actuellement switch entre les serveurs domestiques qu'en créant un nouveau compte.

Je recommanderais l'un des serveurs domestiques suivants. Alternativement, vous pouvez choisir un homeserver vous-même. Les homeservers de cette liste devraient convenir.

Nom du serveur Domaine Description
Matrix Server matrix.org
  • Serveur officiel de Matrix
  • Garantie de mise à jour
  • Parfois laggy parce qu'il a trop d'utilisateurs
Serveur NLtrix nltrix.net
  • Serveur néerlandais gratuit très performant
  • Très rapide pour les résidents des Pays-Bas
Serveur KDE kde.modular.im
  • Propulsé par les dons d'une communauté
  • Se soucie vraiment de la vie privée

Étape 3 : créer un compte

Installez l'application que vous avez choisie à l'étape 1. Assurez-vous de choisir votre bon serveur domestique avant de créer votre compte. Par exemple, si vous choisissez le serveur NLtrix à l'étape 2, assurez-vous de créer un compte sur le domaine nltrix.net. La plupart des applications ont matrix.org par défaut.

Étape 4 : connectez-vous à vos amis

Si vous avez besoin d'aide, rejoignez le salon #matrix:matrix.org. Ici, des inconnus sur Internet sont heureux de vous aider à avancer. En outre, n'hésitez pas à me contacter à @bram:noordstar.me pour obtenir du soutien.

Si un ami vous a envoyé cet article, il a probablement un compte et je suis sûr qu'il sera heureux de vous aider. Envoyez-lui une demande de chat, et vous aurez votre propre message entièrement crypté de bout en bout avec votre ami !
